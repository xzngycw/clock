# 音效文件说明

此目录应包含以下音效文件（.mp3 格式）：

## 必需的音效文件

1. **correct.mp3** - 答对题目的音效
   - 建议：清脆的铃声或"叮"的声音
   - 时长：约 0.5-1 秒
   - 示例：https://freesound.org/

2. **wrong.mp3** - 答错题目的音效
   - 建议：低沉的"嗡"声或温和的提示音
   - 时长：约 0.5-1 秒

3. **click.mp3** - 按钮点击音效
   - 建议：轻快的点击声
   - 时长：约 0.2-0.3 秒

4. **success.mp3** - 成功完成的音效
   - 建议：欢快的胜利音乐
   - 时长：约 1-2 秒

5. **complete.mp3** - 完成所有练习的音效
   - 建议：更长的胜利音乐或欢呼声
   - 时长：约 2-3 秒

6. **tick.mp3** - 时钟滴答声（可选）
   - 建议：轻柔的滴答声
   - 时长：约 0.5 秒
   - 用于时钟运行时的氛围音效

7. **cheer.mp3** - 欢呼音效
   - 建议：孩子们的欢呼声或欢快的音乐
   - 时长：约 1-2 秒

8. **star.mp3** - 星星闪烁音效
   - 建议：清脆的铃声或闪烁声
   - 时长：约 0.3-0.5 秒

9. **transition.mp3** - 页面过渡音效
   - 建议：柔和的过渡音
   - 时长：约 0.3-0.5 秒

## 推荐的音效资源网站

- [Freesound](https://freesound.org/) - 免费音效库
- [Zapsplat](https://www.zapsplat.com/) - 免费音效和音乐
- [Mixkit](https://mixkit.co/free-sound-effects/) - 免费音效
- [Pixabay](https://pixabay.com/sound-effects/) - 免费音效

## 使用说明

1. 下载或创建上述音效文件
2. 确保文件格式为 .mp3
3. 将文件放置在 `public/sounds/` 目录下
4. 文件名必须与上述列表完全匹配

## 临时方案

如果暂时没有音效文件，应用仍可正常运行，只是不会播放声音。
系统会在控制台输出警告信息，提示音效文件缺失。
